# Udith Babu K N - Personal Portfolio

Hey! Welcome to the repository for my personal portfolio and blog. 

I'm a Digital Marketing Expert, and this website acts as a central hub for my projects, my professional experience, and a bunch of digital marketing blogs I've written.

## What's in here?

- **Apple-Style Glassmorphism UI:** The entire design features a pure, high-end Apple-style glassmorphism aesthetic. All components, from interactive cards to mobile nav bars and individual blog layouts, utilize beautiful, highly transparent frosted glass effects that let the vibrant static background shine through.
- **The Blog:** I've written over 100 marketing articles—covering everything from SEO basics to complex B2B sales workflows—which are all baked right into the site as static HTML for lightning-fast loading.

## Tech Stack

I kept the stack modern, clean, and highly performant:
- **React + Vite** for the frontend framework and fast local builds.
- **Vanilla CSS** with a heavy reliance on CSS variables for consistent glassmorphism effects and seamless light/dark theme switching.
- **Phosphor Icons** for clean, minimal iconography.
- **Static HTML** generation for all the blog pages, ensuring optimal SEO and fast delivery.

## Running it locally

If you want to spin this up on your own machine to poke around:

1. Clone the repo
2. Run "npm install" to grab the dependencies
3. Run "npm run dev" to start the Vite development server
4. Open up "http://localhost:5173" in your browser

## Project Structure

- /src: All the React components (Sidebar, Blog, MobileHeader, etc.) and the main CSS styles.
- /public: Static assets, background images, and the individual statically generated HTML blog pages.
- /src/data: Contains the data arrays for the blog posts and other site content.
